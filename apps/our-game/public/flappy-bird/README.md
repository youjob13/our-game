# Flappy Bird Assets

This directory contains the assets for the Flappy Bird game.

## Required Assets

Please download the assets from the original repository and place them in the following structure:

### Images (`img/` directory)

- `BG.png` - Background image
- `ground.png` - Ground sprite
- `toppipe.png` - Top pipe sprite
- `botpipe.png` - Bottom pipe sprite
- `getready.png` - "Get Ready" screen
- `go.png` - "Game Over" screen

### Bird Animations (`img/bird/` directory)

- `b0.png` - Bird frame 0
- `b1.png` - Bird frame 1
- `b2.png` - Bird frame 2

### Tap Animation (`img/tap/` directory)

- `t0.png` - Tap frame 0
- `t1.png` - Tap frame 1

### Sound Effects (`sfx/` directory)

- `start.wav` - Start game sound
- `flap.wav` - Flap sound
- `score.wav` - Score sound
- `hit.wav` - Hit pipe sound
- `die.wav` - Death sound
